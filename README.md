# 관통 Project
: 예적금 상품 비교 및 추천, 커뮤니티 사이트

## Introduction
- 주제 : 예적금 상품 비교 및 추천 사이트
- 기간 : 2023.05.17(수) ~ 2023.05.25(목)
- Team: 서이현, 신성환, 방정우

## 서비스 대표기능
- 예적금 상품 목록 조회 및 상세 페이지 조회
- 상품 상세 페이지의 리뷰(댓글)기능
- 예적금 추천 알고리즘 통한 추천 서비스
- 사용자 프로필 및 사용자 맞춤 상품 추천 받기
- 정보 공유를 위한 커뮤니티
